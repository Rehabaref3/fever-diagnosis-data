# Nooran Individual Contribution

## Summary
My individual contribution focused on improving two visualisations from the Fever Diagnosis and Medicine project. I worked on **Figure 4** and **Figure 5**, which represent **Fever Severity** and **Heart Rate Distribution by Fever Severity**. In the improved version, Figure 4 was redesigned as a **Pareto chart**, while Figure 5 was improved by adding **raw data points** and **mean values** to the box plot. These changes made the graphs clearer, more informative, and more suitable for academic presentation.

In the group project, I also contributed to **System Requirements, Methodology, Introduction to Graphical Analysis, Key Findings, Figure 4 Analysis, and Figure 5 Analysis**.

## How to Run the Program
To run the program, make sure that the Python file and the dataset file `FeverDataset1.csv` are saved in the same folder. Open the folder in Visual Studio Code or another Python environment, then run the Python file. Make sure the required libraries are installed, especially `pandas`, `matplotlib`, and `tkinter`. After running the file, the dashboard window will appear with buttons to open Figure 4 and Figure 5. Each graph will open in a separate window, and the program can be closed using the Exit button.
