# Fever Diagnosis and Medicine Recommendation Using Data Analysis

## Project Summary
This project was developed as part of the **Computer Science Workshop** module. The main aim of the project is to analyse a healthcare-related dataset about fever and patient condition using Python data analysis and visualisation techniques.

The dataset used in this project contains information about patients, including variables such as **age, heart rate, fever severity, and previous medication**. By analysing these variables, the project explores patterns and relationships that help provide a clearer understanding of patient health data.

The group report explains the background of the topic, the dataset description, the system requirements, the system design, the methodology, the graphical analysis, the project results, and the final evaluation. The presentation summarises the main parts of the report and highlights the most important findings using visual graphs and structured explanation.

## Visualisations
The project includes several visualisations to explore the dataset, such as:
- Previous Medication Distribution
- Age Distribution of Patients
- Age vs Heart Rate
- Fever Severity Levels
- Heart Rate Distribution by Fever Severity

In addition to the original group graphs, improved individual visualisations were also developed by group members in separate sections of the repository.

## How to Run the Program
To run the program, make sure that the Python file and the dataset file `FeverDataset1.csv` are saved in the same project folder. Open the folder in **Visual Studio Code** or any Python environment that supports Tkinter. Before running the code, ensure that Python is installed together with the required libraries, especially `pandas`, `matplotlib`, and `tkinter`.

After opening the project folder, run the main Python file. Once the program starts, the dashboard window will appear with buttons that allow the user to open the available graphs. Each graph will open in a separate window for viewing and analysis. The user can move between figures using the available buttons and close the program using the **Exit** button.
